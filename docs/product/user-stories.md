# User Stories

## Inventory
- As a user, I can add an ingredient with quantity and expiry date
- As a user, I can edit or remove inventory items
- As a user, I can photograph my fridge and have ingredients detected automatically (V2)
- As a user, I receive a warning when items are close to expiry

## Recipes
- As a user, I see recipes I can make with what I currently have
- As a user, I can filter recipes by cuisine type and dietary restrictions
- As a user, I can adjust serving size and see scaled ingredient amounts
- As a user, accepting a recipe automatically deducts used ingredients

## Profile
- As a user, I can set dietary restrictions that filter all recipe suggestions
- As a user, I can set my cooking skill level to receive appropriate recipes
- As a user, I can specify what equipment I have (oven, air fryer, etc.)
