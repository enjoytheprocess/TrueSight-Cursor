# Feature design specs

Create one markdown file per feature, e.g. `FEAT-inventory.md`, using [`../templates/FEATURE_SPEC_TEMPLATE.md`](../templates/FEATURE_SPEC_TEMPLATE.md).

Set `spec_link` in `.awp-workspace/2-build/WORK_QUEUE.yaml` to the repo-relative path (e.g. `docs/design/features/FEAT-inventory.md`).
