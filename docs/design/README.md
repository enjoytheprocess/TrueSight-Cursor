# Feature & decision design

Place durable design artifacts here so they are easy to find with `@docs/design/`.

- **Features:** `features/<feature-id>.md` — behavior, APIs, data, acceptance notes
- **Decisions:** `decisions/ADR-YYYYMMDD-NN-title.md` — architecture and contract choices

When an ADR or spec changes build/verify expectations, update `.awp-workspace/3-verify/TRACEABILITY_MATRIX.yaml` in the same change set.
