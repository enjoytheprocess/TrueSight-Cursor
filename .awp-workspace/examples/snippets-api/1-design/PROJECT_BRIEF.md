# Project Brief

Project name: Snippets API
Problem: Developers need a simple, shareable store for code snippets.
Scope: REST API — create a snippet (POST /snippets), retrieve by ID (GET /snippets/:id), delete by ID (DELETE /snippets/:id).
Out of scope: user accounts, search, expiry, tagging in the first two slices.
Primary owner: Dev Team
Primary component: api
