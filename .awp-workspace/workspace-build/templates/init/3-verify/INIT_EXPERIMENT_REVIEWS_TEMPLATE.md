# Experiment Records

Agent-maintained record of experiment plans and outcomes during Build/Verify.
One entry per task that uses `Specialist Advisor = experiment`.

Add entries using `3-verify/templates/EXPERIMENT_PLAN_TEMPLATE.md`.
Set `advisor_status: complete` in `1-design/TASK_READINESS.yaml` once the plan is documented and the decision is recorded.
If the experiment cannot proceed as designed, record the blocker in `3-verify/GAPS_AND_DEVIATIONS.yaml`.
