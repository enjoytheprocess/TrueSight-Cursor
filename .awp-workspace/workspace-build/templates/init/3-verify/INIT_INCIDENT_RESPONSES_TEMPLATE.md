# Incident Response Records

Agent-maintained record of incident analyses during Build/Verify.
One entry per task that uses `Specialist Advisor = incident`.

Add entries using `3-verify/templates/INCIDENT_RESPONSE_TEMPLATE.md`.
Set `advisor_status: complete` in `1-design/TASK_READINESS.yaml` once the incident is contained enough to proceed safely.
Record corrective action tasks in `2-build/WORK_QUEUE.yaml` and link them from the entry.
If root cause or remediation is unclear, record it in `3-verify/GAPS_AND_DEVIATIONS.yaml`.
